Mac Install Instructions

  1. Download the zip file containing the application.
  2. Extract the application from the zip file. (Double-clicking the zip file in Finder should accomplish this.)
  3. Move the application to your Applications folder.
  4. Run the application.
NOTE: The first time running the application a message may appear preventing you from opening the application. This is a message from Gatekeeper, a Mac security feature. It appears because the application is not signed by a registered Apple developer (I did not want to pay the $99 fee to register in the program). The easiest workaround to open is to right-click on the application and select "Open" from the context menu. This article has more information.
